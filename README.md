# Van-Buren-Sounds

Sounds from Van Buren (cancelled Fallout 3)

## Installation

### Low quality
1. Download LQ version from the [latest release page](https://github.com/egornovivan/Van-Buren-Sounds/releases/latest).
1. Extract into `data\sound\music`.
1. In `fallout2.cfg` find `[sound]` section and set `music_path1=data\sound\music\`.

### High quality
1. Make sure you have [sfall](https://github.com/phobos2077/sfall/) (`ddraw.dll`) version no less than 4.2.9.
1. Download HQ version from the [latest release page](https://github.com/egornovivan/Van-Buren-Sounds/releases/latest).
1. Extract into `data\sound\music`.
1. In `fallout2.cfg` find `[sound]` section and set `music_path1=data\sound\music\`.

### Ultra high quality
1. Make sure you have [sfall](https://github.com/phobos2077/sfall/) (`ddraw.dll`) version no less than 4.1.1.
1. Download UHQ version from the [latest release page](https://github.com/egornovivan/Van-Buren-Sounds/releases/latest).
1. Extract into `data\sound\music`.
1. In `fallout2.cfg` find `[sound]` section and set `music_path1=data\sound\music\`.
1. In `ddraw.ini`, find `[Sound]` section and set `AllowDShowSound=2`.
